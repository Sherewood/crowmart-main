How to work:
1.if not already installed, install node.js onto system.
2.inside /crowmart-main, open your command window and enter 'npm install'. This will install all needed dependencies.
3.In the same command window, enter 'node server.js'. You should see 'Server running at http://127.0.0.1:3001/' if it is working.
4.enter 'http://127.0.0.1:3001/' into your browser of choice and the website will open up with the proper css and working cart/add to cart functionality.
